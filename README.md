# smtp_send


**使用本脚本时，你需要以下内容** 


| variate      | typeof |
| --------- | -----:|
| 邮件服务器地址  | String |
| 端口     |   Numbers |
| 账号      |   String |
| 密码      |   String |
| 发件人地址      |   String |
| 收件人地址      |   String |
| 标题      |   String |
| 正文      |   String |

# 开始使用
启动命令

    >python smtp_send.py 邮件服务器地址 端口 账号 密码 发件人地址 收件人地址 标题 正文
